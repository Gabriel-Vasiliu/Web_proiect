Scop

Scopul acestui document este de a oferi o descriere aprofundata a proiectului Collecting Bottles on Web. Acest document va explica caracteristicile si scopul acestui proiect.

Domeniul produsului

Collecting Bottles on Web este un site web destinat colectionarilor de recipiente.

Descriere generală

Perspectiva produsului

Collecting Bottles on Web este un site web destinat colectionarilor de recipiente utilizatorii autentificati vor putea crea, inventaria si partaja (sub)colectii de recipiente in functie de diverse caracteristici (tip, imagine, valoare, tara, perioada de utilizare, istoric, existenta etichetei etc.), plus vor putea importa/exporta datele referitoare la recipientele detinute.

Clasele și caracteristicile utilizatorilor

Utilizatori obișnuiți care nu au nevoie de niciun fel de autentificare și doresc să vizualizeze top-ul oferit de site. Utilizatorii logati pot sa-si gestioneze colectiile.

Mediul de operare

Acest site web poate fi găzduit pe orice tip de sistem de operare care poate rula PHP: sisteme de operare asemănătoare Unix, Windows și macOS. Site-ul poate fi accesat de pe orice dispozitiv, inclusiv mobil, care are un browser web modern.

Constrângeri de proiectare și implementare

Acest software este dezvoltat folosind PHP, HTML și CSS. Datele sunt stocate într-o bază de date MySQL. Proiectul este structurat folosind modelul de proiectare model-view-controller design pattern.

Dependențe

Site-ul depinde de PHP 8 și de un server web precum Apache.

Cerințe de interfață externă

Interfețe software

Acest instrument necesită pe partea serverului PHP 8 cu o baza de date MySQL.

Interfete de comunicare

Paginile web sunt transferate prin HTTP/HTTPS. Pentru autentificare este de preferat HTTPS pentru a preveni atacurile.

Caracteristicile sistemului

Prezentarea produsului

Pe pagina de start este prezentată scopul acestui produs.

Autentificare

Utilizatorul își poate introduce username-ul și parola. După apăsarea butonului de autentificare serverul le va valida solicitarea și, dacă reușește, raman autentificați pe toată durata sesiunii.

Alte cerințe

Cerinte de performanta

În ceea ce privește memoria, acest instrument necesită cel puțin 512M de RAM

Atribute de calitate software

Pe lângă diferitele modalități de vizualizare a datelor, site-ul este responsive și poate fi utilizat pe dispozitive cu ecrane mici precum tablete sau smartphone-uri.
